=== WP Disable Posts ===
Tags: post
Requires at least: 3.0
Tested up to: 3.5
License: GPLv3

This plugin disables the built-in WordPress Post Type `post`

== Description ==

Apparently not everyone will utilize the built-in WordPress Post Type `post`. This plugin disables it completely.

== Installation ==

Upload the WP Disable Posts plugin to your blog, Activate it.
